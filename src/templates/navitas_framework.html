Project pitch

Navitas Framework is a plug-in for Android Studio that allows you to track the power consumption of various components of an Android device, such as the screen, CPU, as well as Wi-Fi and Bluetooth, which are to be integrated within the framework of this project. With its help, developers of Android applications will be able to find out how much energy in mA a particular module has consumed while the application is running, and, if necessary, start optimizing it.

According to the analysis of the subject area, it was found that most competitors designed for the latest versions of Android give only approximate time estimates for power consumption, and those that calculate it in mA are hopelessly outdated.
